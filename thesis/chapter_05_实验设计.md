# 第五章 实验设计

建议字数：约 1500 字。

本章重点是说明实验怎么做，而不是提前分析结果，因此篇幅可以适中，要求结构清楚、变量明确、对比关系完整。

本章用于说明“你准备如何验证方法是否有效”，要把实验对象、对比方式、评测指标和实验组织方式交代清楚。

本章建议重点阐明以下内容：

- 实验目标的分层设计，例如离线训练验证、在线对局验证、基线对比、消融分析、参数或预算敏感性分析等。
- 所使用的游戏环境、数据集来源、训练样本构造方式及相关配置。
- teacher-only、residual_v1、旧版 baseline value 模型等不同模型或方法的实验角色。
- 对比基线的设置，包括 `pure_mct`、`heuristic_mcts`、早期 value 模型以及不同集成配置。
- 在线 benchmark 的设计方式，例如 fixed-sims、fixed-time、多 seed、多游戏平均等。
- 离线指标与在线指标分别如何定义，为什么这些指标能够支撑论文结论。
- 消融实验或补充实验的设计逻辑，例如 dataset size、integration 方式、selective 配置等。
- 实验实现中的控制变量与公平性说明。
